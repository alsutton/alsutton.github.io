Android 4.0.3 - Why .3 Is Important

  • Sat, Dec 17, 2011
  • One-minute read

Yesterday Jean-Baptiste Queru announced that Android 4.0.3 had been open sourced via the Android Open Source Project, which is bigger news than you might think.

Versions before 4.0.3 were focused on the Galaxy Nexus and its hardware. 4.0.3 is a multi-device release which supports the Nexus S, Xoom, and should be used as the basis for other ports.

So if you see an Ice Cream Sandwich / Android 4.0 device you should look at the specific version it’s running, if it’s not a Galaxy Nexus and it’s running something before 4.0.3 then you might want to ask when it’ll get the update.
