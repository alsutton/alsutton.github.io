Wireless Broadband for a SOHO Using a Huawei E220 and a Router

  • Tue, Jan 8, 2008
  • One-minute read

I’ve heard a few people are looking for a solution to using a cellular modem for their small or home office.

The solution I’ve used is based on the Draytek Vigor 2800 and the Huawei E220 USB modem. The instructions on configuring these two can be found on Drayteks' site at http://www.draytek.co.uk/support/kb_vigor_3g.html
